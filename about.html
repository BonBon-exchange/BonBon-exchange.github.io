What is BonBon?

BonBon is a browser offering an innovative interface and privacy. Nothing is collected, we don't have any database nor any backend. We use Nucleus.sh to provide us analytics about app usage and app errors.

Features

Collections: create a composition of webviews

Bookmarks

Browsing History

Download manager

A bunch of settings

Snap windows to the left or to the right dragging it to the edge of the app

Darkmode

Drag and reorder tabs

Download extensions from Chrome Web Store

uBlockOrigin by default

Keyboard shortcuts: ctrl (+ shift) +t, +r, +w | alt + scroll
